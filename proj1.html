House Of Pet

House of Pet is an animal grooming queuing system using midtrans payment gateway. The technology used is Laravel with boostrap front-end. This project is done in a group and the final project from the workshop course in semester 4.

The background of this project is that House of Pet Partners on Soekarno Hatta Street find it difficult when do grooming queue orders. This project was made to overcome this problem by creating this grooming queue system.

In this project i worked on features:
1. Queuing system backend
booking logic and queuing logic
2. Admin actor CRUD feature
Features to add selected package categories and Features to manage bookings and queues

Processing Time: February 2023 - June 2023
