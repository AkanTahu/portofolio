SIMAPRES

SIMAPRES is a ranking system for Polinema's best students every year using a mobile platform and admin using the website. This project is the final group project of basic mobile programming lecture material and decision support systems in semester 5.

The flow of this system is that students are given time to create an account first and after that input the grades and achievements obtained. After that the admin validates the values and achievements that students get. After the validation and ranking time is complete, the rankings of all students will appear.

The ranking criteria here are assessed from GPA, the number of alpha attendance, and achievements. These criteria will be calculated using the EDAS method through flask so that it can be updated.

In this project i worked on features:
1. Mobile Backend and Frontend
Create CRUD features on student actor accounts, create ranking features (all students or majors)
2. Backend and Frontend Website Admin
Create a CRUD feature to add student data and create a student achievement check feature
3. Backend Decision Support System
Create a flask-based Decision Support System and use the EDAS method. This flask system was inspired by my friend I Dewa Made
